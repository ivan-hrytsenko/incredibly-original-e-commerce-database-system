# Лабораторна робота №6

## Початковий стан

База даних була імпортована з Docker за допомогою `npx prisma db pull`. Її стурктура складається з 8 моделей, що відповідає результатам абораторної роботи №5.

## Зміни та моделювання

У процесі виконання було модифіковано файл schema.prisma для розширення функціональності системи.

1. **Система відгуків:** Таблиця `reviews` для зберігання відгуків про товари.
    * **Зв'язок:** One-to-Many між `products` та `reviews` (один товар може мати багато відгуків).
2. **Статус активності товару:** До таблиці `products` було додано поле `is_active` (Boolean).

## Міграції
* **add-reviews-table:** SQL-скрипт для створення таблиці `reviews` та визначення Foreign Key для зв'язку з продуктами.
* **add-product-active-status:** SQL-команда `ALTER TABLE` для додавання колонки `is_active` до існуючої таблиці `products`.
